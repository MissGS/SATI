<h1>如何设置为壁纸?</h1><br>
<h3>Lively Wallpaper</h3><br>
<a>1:下载Lively Wallpaper 大佬开发开源免费的(自行下载)</a><br>
<a>2:点击Lively Wallpaper右上角加号 图片示意:https://github.com/user-attachments/assets/1eb23f4d-f306-42ab-8075-c55b3c76102a</a><br>

<a>4:完成</a>
